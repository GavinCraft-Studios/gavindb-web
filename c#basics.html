The C# Basics

In this episode you will learn how to costomize the console and how to work with Console ReadLine and Console WriteLine.

Episode Notes

  • Console.Title = "Example Window Name"; this method changes the name of the console window

  • Console.ForgroundColor = ConsoleColor.Green; use this to change the text color in the console

  • Use Console.ReadLine(); to add a pause between code such as Console.WriteLine(); (remember to hit enter to trigger readline)

  • Use //This is a comment, use this to write a comment in your code and help you organize when things get messy
