Arrays

In this episode we will explore how to store data in arrays and lists.

Episode Notes

When creating a list of items you can use an array. the syntax for this is a bit different but quite simple. As always an example will be below. The string is the type of varriable you would like to store and the favourites is the variable name. The brackets after string is what indicates that its an array. The rest should be self explanitory as you need to set multiple values in the array.

Another type of list is a list... these are used when you don't know how many items will be on the list. Remember that to use a list you must write "using System.Collections.Generic;" at the top of your code. To start a list you write, "List name = new List" to add and remove items from a list you can write, "name.Add(Value);" and "name.Remove(Value);"
