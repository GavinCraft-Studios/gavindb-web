DOT Net

When creating a project remember to:

  1. Create a new folder and in vscode click file, open folder and select your created folder

  2. In vscode go to Terminal, New Terminal this should create a terminal at the bottom of your screen

  3. In the created console type: "dotnet new console" and hit the enter key, this creates files in our project

  4. Click on Program.cs in the file explorer, this is where your code will go!

  5. Use the shortcut: Ctrl+Shift+P to open a searchbar and look for: ".NET: Generate Assets for Build and Debug" click this option

  6. With these newly created files in your system open the folder and go to launch.json

  7. Go to around line 17 and change the "console" option from "internal console" to "externalTerminal"

  8. To run the program go to Run, Start Debugging. Make sure to have Console.ReadLine() at the bottom of your code
