Note that the content in the parenthesis is your condition. If the condition is met the upper code will be executed. If not the lower code will be executed. The else statement if undesired is completely optional. Different symbols are used for different conditions such as != means not equal to, == means equal to, > and >= means greater than and greater than or equal to respectivley.
