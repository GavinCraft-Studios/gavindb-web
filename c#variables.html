Varriables in C#

In this Episode you will learn how to store and use data in Varriables.

Episode Notes

A variable is used to store data. An example of this is using a variable to store the player's name in a game. There are many different types of variables but there are 4 essential types.

  • An Integer or int is used to store a positive or negative whole number, No Decimals / Fractions

  • A Float is used to store a decimal number this again can be positive or negative.

  • A String variable is used to store text, just like our player names example. Quotations are always used with String values.

  • Finally a bool is used when you desire a True or False value.







If you need to write a variable use this to help, (Remember string values are always in "")

Some examples of this is, string userName = Console.ReadLine(); int number = 0;

Convert Variables by, number = Convert.ToInt32("string value"); userName = Convert.ToString(25);

To do math with these variables use, * for multiplication, + and - for addition and subtraction, and / for division
